ThunderMusic

Thunder Music : Le lecteur de musique d'aujourd'hui

Thunder Music est un lecteur de musique ultra léger. Il offre pleins de mini-features comme un youtube-downloader, une synchronisation avec Dropbox, des playlists, et plus ! Il dispose de trois modes, léger (450x200) qui comporte le player et des petits boutons pour les features; normal (redimensionnable) qui comporte un player en bas, et un grand menu de playlist; et fullscreen, un grand lecteur en plein ecran avec le player en bas et des boutons en dessous. Thunder Music est écrit en JavaScript, HTML et CSS. Il utilise Electron de Github. Le projet est Open-Source et actuellement en lourd développement. Vous pouvez retrouver le repo Github ici.

OpenLauncherLib

OpenLauncherLib

L'OpenLauncherLib : Créez votre launcher Minecraft autour de cette librairie !

L'OpenLauncherLib est une librairie Java permettant de coder un launcher de zéro. Elle permet le lancement de n'importe quelle version de Minecraft, simplement. Elle est entièrement personnalisable de font en comble. Elle offre pleins de features souvent présents dans les launchers, elle permet de pouvoir bouger une fenêtre avec la souris en rajoutant 3 lignes, ou de sauvegarder/charger le pseudo avec 3 lignes encore, et l'emplacement du dossier du jeu dynamique est intégré. Avec l'OpenLauncherLib, créer un launcher Minecraft n'a jamais été aussi simple. C'est la première librairie ouvert (LGPL) pour créer son launcher Minecraft. Elle est écrite en Java mais une adaptation C++ est prévue. Comme elle est sous LGPL vous pouvez l'utiliser commercialement sans aucune réstriction. Son repo Github est ici.

S-Update

S-Update

S-Update : La librairie de mise à jour ultime

S-Update est une librairie de mise à jour pour les développeurs. Elle permet la mise à jour simplement et rapidement, de façon sécurisée. Vous pouvez choisir d'utiliser les MD5s pour la sécurité où la date de dernière modification des fichiers pour la rapidité. Elle utilise un serveur en php simple d'utilisation, vous avez juste à mettre vos fichiers dans le dossier files. Dès que vous les modifierais, ils seront mis à jour chez les utilisateurs. En plus de ça, les fichiers intrus ou modifiés chez les utilisateurs, sont supprimés ou retéléchargés. Biensur pour empêcher la suppression d'un fichier ou d'un dossier, mettez son nom dans le fichier su_ignore.idx. S-Update est Open-Source sous license LGPL, çela signifie que vous pouvez l'utiliser de façon commerciale sans aucune réstriction. Son repo Github est ici.

Sender

Sender

Sender : Envoyer des fichiers en local simplement !

Sender est un petit programme très léger est très simple d'utilisation, permettant d'envoyer des fichiers à d'autre ordinateurs en local. Chaque ordinateur a un ID généré en fonction de son IP Locale. Lancez le programme, une petite boite apparait, cliquez sur le plus pour avoir plus de détail (comme votre ID), glissez des fichiers dans la boîte, saisissez l'ID du destinataire (il doit avoir Sender lancé), et pof ! Voilà ! La déstination des fichiers est configurable au lancement. J'avais fait ce programme pour apprendre les sockets à la base, mais j'avais envie de le partager. Il est sous license Apache 2, le repo est ici.
